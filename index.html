Nigeria Data Site Interactive Graphics Demo

A space to demonstrate the functionality and scalability of graphics for the Nigeria PVT data site.

Animations

Space to be used to test out animations for the site.

Voter Turnout is Falling

Time-delayed animation to indicate that turnout is falling, in which voters fall away.

To be placed in block 1 of the turnout page.

Graphs

Some PVT data lends itself well to representation in graphs. These examples are intended to be used on the interactive versions of the site.


Nigerian Turnout Over Time

Simple line graph to show decline in turnout since 2003 in Nigeria.

To be placed in block 2 of the turnout page.



Interactive Pyramid Chart of Accreditation and PVC Use

Percent of polling units in 2015 in which voters were accredited to vote without their PVCs being checked and units in which voters were accredited even though the card reader could not read their PVC.

To be placed in block 4 of the technology page.

Comparison Between 2011 and 2015 Turnout Figures

A toggle button to allow a user to compare the 2011 and 2015 PVT estimates to the INEC results.

To be placed in block 1 of the PVT page.

Maps

These maps or combination thereof could be used to display geographic PVT data over time.

Interactive Turnout Map

An interactive map to show turnout in 2015. On a desktop a user could mouseover the state SVG objects to reveal a tooltip with the turnout in 2015. On a mobile device, clicking on the state would reveal the tooltip. If we went this route, we could use a GIF as a replacement image by calling the map with the object class. See the commented out code in the source.
